# MERO Marketing

The marketing engine of **Merowingus Studio** — a set of reusable, AI-powered **agents** that do
marketing for any product: video scripts, community posts, launch kits, SEO briefs, funnels, and
funding outreach.

> A parallel sibling tool under MERO Studio (alongside MERO SEO, MERO Job Hunt). Self-use first
> (promote Divergentum + the studio), then a dual-use sellable tool for founders.

## The idea

**The machine, not the strategy.** We wrote marketing strategies by hand once (the Divergentum docs).
From them we extracted reusable agents. Every next product gets marketing by *running the agents* —
parametrized by {product · ICP · channel} — instead of starting from a blank page.

## Quick start

Open this folder in Claude Code and run an agent, e.g. `/script-writer` → give it a product + its
wow-moment → get ready-to-shoot Reels/Shorts scripts.
