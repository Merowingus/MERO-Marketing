# X / Twitter agent

Build-in-public + launch announcements for the indie/dev/AI crowd. Output → `../../../channels/x-twitter/`.
Tags/keywords: `../../../campaigns/<x>/keywords-tags.md`.

## Channel norms
- First line is the whole game — a strong hook tweet earns the open.
- Threads: one idea per tweet; show, don't tell (screenshots/clips of the wow).

## Method
1. **Hook tweet** (the pain or the surprising result).
2. **Thread:** one beat per tweet (story → the work → the wow → CTA). Reuse blog/Reddit material.
3. **Single posts:** launch announce, a clip, a milestone. One link with UTM (in reply, not first tweet).

## Output → `channels/x-twitter/`
Hook + thread + single-post variants.

## Rules
- Hook first. Honest, specific, first-person. No engagement bait.
- UTM (`utm_source=x`). Log in `../../../experiments.md`.
