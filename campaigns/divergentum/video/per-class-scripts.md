# Divergentum — Per-class promo clips  [Reels / TikTok · solo-D&D players]

Format per clip (15–30s): **screen-recording of the interface + the real-time generated cinematic
scene** for that class's signature move. Hook first (the class fantasy / pain), then the wow scene.
Zero fabrication — only show abilities the game actually has.

> On-screen text/voice below is in RU (Mykola's language). For reach on IG/TikTok in the D&D niche,
> swap to EN — the shot list stays identical. One clip → Reels + Shorts + TikTok.
>
> **Tags = the SEO semantic core**, not decoration. Pull every hashtag from
> [`../keywords-tags.md`](../keywords-tags.md). Default per class: #dnd #dnd5e #solorpg
> #aidungeonmaster #divergentum #&lt;class&gt; (+ swap in niche tags like #soloroleplaying #dndsolo).

```

  Развитие:

  #dnd #dnd5e #solorpg #aidungeonmaster #divergentum #<класс>
```

## 🎬 Rogue — Sneak Attack · ~20с
- **Хук (0–2с):** текст «POV: ты бьёшь первым — из тени, пока он тебя не видит» · тёмный кадр переулка
- **Развитие:**
  - Визуал: печатаю в чат «подкрадываюсь со спины и бью кинжалом, пока он отвлёкся»
  - Текст на экране: «Cunning Action → скрытность → Sneak Attack»
  - Звук: тихие шаги → бросок d20 (ASMR) → резкий удар
  - Показ: проверка Stealth по 5e → ИИ-Мастер описывает внезапную атаку + бонус урона Sneak Attack
- **Wow:** сцена генерится — фигура в капюшоне, кинжал входит из темноты, факелы, кровь на камне
- **CTA:** «Мастер, который не галлюцинирует, а водит по правилам. 70 ходов бесплатно — ссылка в профиле»
- **#:** #dnd #dnd5e #solorpg #rogue #aidungeonmaster #divergentum

## 🎬 Fighter — Action Surge · ~20с
- **Хук (0–2с):** текст «POV: один раунд — две атаки. Враг не успел моргнуть» · кадр поля боя
- **Развитие:**
  - Визуал: печатаю «бью, а потом сразу бью ещё раз, пока есть импульс»
  - Текст на экране: «Action Surge → лишнее действие в раунде»
  - Звук: лязг стали → двойной удар → героический хор
  - Показ: ИИ-Мастер разрешает Action Surge по 5e → две атаки в одном ходу, броски на попадание
  - (по ситуации: Second Wind — подлечился посреди боя; Fighting Style — бонус стиля)
- **Wow:** сцена генерится — воин в гуще схватки, две молниеносные атаки, искры от меча
- **CTA:** «Строгие правила 5e + сцена каждого боя в реальном времени. 70 ходов бесплатно»
- **#:** #dnd #dnd5e #solorpg #fighter #aidungeonmaster #divergentum
