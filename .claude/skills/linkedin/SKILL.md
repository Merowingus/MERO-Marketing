# LinkedIn agent

The personal-brand channel — Mykola as the AI PM who builds in public (ICP A: founders/builders).
Output → `../../../channels/linkedin/`. Voice: studio `strategy/positioning.md`.

## Channel norms
- First 2 lines show before "…see more" — front-load the hook.
- Practical > inspirational. A concrete lesson + a real artifact (screenshot/number) beats a pep talk.
- Plain text, short paragraphs, one clear takeaway. Link in a comment, not the post body.

## Method
1. Pull a real lesson from the work (the dev journey, a launch result, a tool built).
2. Draft: hook (2 lines) → the story/work → one takeaway founders can use → soft CTA.
3. Tie to the AI-PM wedge (positioning). Reuse the blog post, sharpened for LinkedIn.

## Output → `channels/linkedin/<topic>.md`
Post text + the link (UTM, for the first comment).

## Rules
- Honest, first-person, no humble-brag. One takeaway per post.
- UTM (`utm_source=linkedin`). Log in `../../../experiments.md`.
